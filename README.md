# Weather App

A responsive weather application built with vanilla JavaScript and the OpenWeatherMap API.

## Features
- Search current weather conditions by city
- Displays temperature, weather description, humidity, and wind speed
- Live weather icons
- Error handling for invalid city names

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
